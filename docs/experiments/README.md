# experiments

スライド生成品質の改善ログをここに保存します。

運用ルール:

- 品質改善、prompt 調整、renderer 変更、export 挙動変更、入力体験の変更を行った場合は原則ここに 1 本ログを残します
- 1テーマ 1ファイルを基本にし、日付とテーマをファイル名に含めます
- Before / After を比較できる固定入力やスクリーンショット、生成 JSON がある場合は `docs/experiments/artifacts/` に保存してリンクします

命名例:

- `2026-04-04-prompt-structure-tightening.md`
- `2026-04-05-summary-slide-stability.md`

作成時は [`docs/EXPERIMENT_LOG_TEMPLATE.md`](/home/sora/dev/llm-apps/slide_app/docs/EXPERIMENT_LOG_TEMPLATE.md) を複製して使ってください。
