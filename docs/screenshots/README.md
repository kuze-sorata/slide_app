# Screenshot Notes

This directory is reserved for portfolio screenshots for `slide_app`.

Recommended files:

- `input_en.png`: English input page
- `preview_en.png`: English preview page after generation
- `json-editor.png`: JSON review/edit area

Capture guidance:

- use either mock mode or a configured API provider
- prefer one short English business request with 5 slides for hiring-oriented screenshots
- avoid exposing local file paths, tokens, or personal data
- keep browser zoom and window size consistent across images

Suggested capture flow:

1. Start the app locally
2. Open the form at `http://127.0.0.1:8000/`
3. Switch the UI to `EN`
4. Generate a deck from a short English sample request
5. Save 1 to 3 screenshots for the README
