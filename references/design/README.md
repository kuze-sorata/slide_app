# Design References

Canva などの参考テンプレート画像やスクリーンショットをここに保存します。

命名例:

- `canva-exec-summary-01.png`
- `canva-problem-solution-01.png`
- `canva-progress-report-01.png`

運用ルール:

- 元画像はこのフォルダに置く
- まずは PNG / JPG / WebP をそのまま保存し、元 URL は別途メモする
- 画像ごとの観察メモは [`docs/design_guidelines.md`](/home/sora/dev/llm-apps/slide_app/docs/design_guidelines.md) に追記する
- 完全コピーではなく、再利用したいルールだけを抽出する
- prompt / preview / PPTX へ反映する時は、このプロジェクト向けの設計ルールへ言い換える
- 共通化できたルールは [`docs/slide_patterns.md`](/home/sora/dev/llm-apps/slide_app/docs/slide_patterns.md) に昇格させる
